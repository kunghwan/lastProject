# 오늘 할일

1. 맡은 페이지 중 작업할 페이지 하나 선택해서 README.md 만들고 작업
2. 작업 끝나면 PR 날리고 팀장한테 리뷰 신청하기
3. 리뷰 다같이 진행할 수 있도록 7교시에 진행
4. 리뷰 후 문제 없으면 MERGE

## github branch 전략

1. 본인의 branch에만 push 하기! **팀장포함!예외없음**
2. merge 된 후 본인 브랜치에서 add&commit 후 gitpull origin main
